# E-commerce-001-GBG

Site e-commerce (Commerce électronique) — modèle **Orfarm** en HTML/CSS/JS.

## Structure

- `orfarm/` — Le site e-commerce (pages HTML, assets CSS/JS/images)
- `documentation/` — La documentation du template

## Lancement en local

Le projet est constitué de pages HTML statiques. Ouvrez simplement `orfarm/index.html` dans un navigateur,
ou servez le dossier via un serveur web (WAMP, Live Server, etc.).
